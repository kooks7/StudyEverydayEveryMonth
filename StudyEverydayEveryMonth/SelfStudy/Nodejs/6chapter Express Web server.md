# 1.1  Express-generator 로 빠르게 설치하기

* app.js 핵심적인 서버 역할
* www 서버를 실행하는 스크립트

* routes 주소별 라우터들을 모아둔 곳.
* views 폴더는 템플릿 파일을 모아둔 곳.
